# DreamDiary

### 드림 다이어리(Dream Diary)


 꿈을 기록하는 다이어리, 드림 다이어리(Dream Diary)입니다. 제목, 날짜, 기분, 내용을 선택하여 꿈을 저장할 수 있습니다! Android Studio로 구현한 프로젝트입니다.
